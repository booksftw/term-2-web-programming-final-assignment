# Movie Search App
This is a web application that allows you to search for movies and view details about them.
## Features
* Search for movies by title, year, runtime greater, or runtime less.
* View movie details in cards.
* Paginate through movies to find the one you like!
## Technologies Used
* Node.js
* Express.js
* EJS (Embedded JavaScript) templating engine
* MongoDB
* Mongoose ODM
* Tailwind CSS for styling
## Getting Started
1. Clone the repository to your local machine. https://github.com/booksftw/term-2-web-programming-final-assignment.git
2. Install the required packages using npm install.
3. Start the application using npm start.
4. Open your browser and navigate to http://localhost:5001 to view the application.
